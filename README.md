# Rule-Based-AI-Chat-Assistant
This is a rule-based "AI CHATBOT" built in Python to chat in a friendly manner with the user and also help them with some basic questions.

This mini-project is built using Python and modules like "datetime" and "time".

The code at first greets the user according to the current time using the "now.hour" function of the "datetime" module.
It consists of a dictionary to store the expected chats from the user and their responses by the bot.
It consists responses like "Hello","how are you?",etc.
The code exits if the user writes "bye".
